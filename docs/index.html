URL Redirector

A simple, privacy-focused Chrome Extension to redirect URLs and improve focus.

Download on Chrome Web Store View on GitHub

Features

  • Universal Redirection: Redirect any URL to a target URL.
  • Privacy First: No tracking, analytics, or data collection.
  • Lightweight: Uses Chrome's native declarativeNetRequest API for zero-latency redirects.

Permissions & Privacy

Last Updated: November 2025

Why do we need "Read and Change Data" permissions?

Because to redirect a page, the extension must be able to "see" the request and "change" it to the new URL. We do not store or read your actual page content.

Our Data Policy

The URL Redirector extension is designed with privacy as its core principle.

  • No Data Collection: We do not collect, store, or transmit your browsing history, personal information, or usage data.
  • Local Storage: Your redirect rules are stored locally on your device using Chrome's storage API.
  • No Third Parties: This extension does not communicate with external servers.
